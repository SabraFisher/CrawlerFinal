
# Dungeon Crawler

A console-based role-playing game developed in C# (.NET 9).

## Features
- Player character progression and experience (XP)
- Turn-based combat mechanics
- Various hostile creatures to battle

## Getting Started
To get started with the Dungeon Crawler game, follow these steps:

1. Open the project in Visual Studio 2022.
2. Set `DungeonCrawlerG2` as the startup project.
3. Build and Run the project to start the game.

## Technologies Used
This project utilizes the following technologies:
- C# 13.0
- .NET 9.0
